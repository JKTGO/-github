Native Android application featuring:

- Target SDK: API 34 (Android 14)
- Kotlin dengan MVVM Architecture
- Version: 1.0 (Build 1)

2. **Hotel Recommender:**
   - Rekomendasi hotel berdasarkan rating bintang

**Key Features:**
